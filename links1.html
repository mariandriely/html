O que são os links em HTML5?

HTML links são "super-links" (do inglês,hyperlinks).
Você pode clicar em um link e pular para outro documento.
Quando você movimenta o mouse sobre um link, sua seta irá se tornar uma pequena mão.
Isso significa que essa é uma área sensível ao mouse, algum evente irá ocorrer.

HTML links são definidos com a tag ‹a›

Exemplo:
 ‹a href="https://www.w3schools.com" ›Isso é um link‹⁄a›
Resultado:
 Isso é um link
