Elementos HTML

Um elemento HTML basicamente consiste no início/final de uma tag, com o conteúdo inserido, respectivamente.

Exemplo:
‹tagname›Conteúdo‹⁄tagname›

Elementos aninhados
Elementos HTML podem estar aninhados (elementos podem conter elementos)
Todos documentos HTML consistem em elementos aninhados.

Exemplo:

‹!DOCTYPE html›
‹html›
 ‹body›
   ‹h1›Sou um heading‹/h1›
 ‹/body›
‹/html›
html contém { body/h1 }
body contém h1.
